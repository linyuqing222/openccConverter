# OpenCC 中文簡繁轉換

為 [NVDA](https://www.nvaccess.org/) 提供離線的簡體 ⇄ 正體中文轉換。

OpenCC 中文簡繁轉換會把**選取的文字**或**剪貼簿的文字**在簡體與正體中文之間轉換，讀出結果並複製到剪貼簿。它**完全離線運作**——所有字典都已內建，不需要網路連線。

## 功能

- 透過指令層轉換**選取的文字**或**剪貼簿的文字**，並可隨時**交換**轉換方向。
- 結果會被**讀出**並**複製到剪貼簿**。
- 可雙向轉換——簡體 → 正體，或正體 → 簡體，包含臺灣標準用字與詞彙層級的轉換——在附加元件設定中選擇。

## 系統需求

- NVDA **2024.1** 以上版本（最後測試版本為 NVDA **2026.1**）。

## 安裝

1. 下載 `openccConverter-<version>.nvda-addon`。
2. 用 NVDA 開啟它（或透過 *NVDA 功能表 → 工具 → 附加元件商店 → 從外部來源安裝*），並依照提示操作。
3. 出現提示時重新啟動 NVDA。

## 使用方式

按 **`NVDA+shift+c`** 開啟指令層（會聽到提示音），接著按一個鍵：

- **`s`**——轉換**選取的文字**。
- **`c`**——轉換**剪貼簿的文字**。
- **`w`**——**交換**轉換方向。

按其他任何鍵都會關閉指令層。轉換後的完整文字會放到剪貼簿。接著結果會被讀出：短結果完整讀出，較長的結果則以字元數回報（這樣龐大的轉換結果就不會被整段念出來）。較長的轉換進行中會反覆發出進度提示音——處理整本書之類的大量輸入時很方便。

交換方向（**`w`**）會報出新的方向；如果你有選取文字，也會用新的方向轉換該選取範圍。你可以在 NVDA 的 *輸入手勢* 對話方塊中、於 **OpenCC 中文簡繁轉換** 類別下重新指派指令層的手勢。

## 設定

在 *NVDA 功能表 → 偏好設定 → 設定 → OpenCC 中文簡繁轉換* 中選擇轉換方向。你的選擇會被儲存，並在重新啟動後保留。在你自行選擇方向之前，預設會依 NVDA 的介面語言而定：簡體中文介面 (zh_CN、zh_SG) 預設為 **正體中文（臺灣）→ 簡體中文（含詞彙轉換）**；香港或澳門介面 (zh_HK、zh_MO) 預設為 **簡體中文 → 正體中文**；其他語言——包括臺灣——預設為 **簡體中文 → 正體中文（臺灣，含詞彙轉換）**，會產生道地的臺灣用語（例如 软件 → 軟體、内存 → 記憶體）。
